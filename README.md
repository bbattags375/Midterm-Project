# Midterm-Project
A desktop GUI application built with Python and Tkinter. Users enter up to 5 test scores and the program calculates their total, average, and letter grade. Empty fields are skipped automatically and invalid input triggers an error popup.
What I Learned

Building GUI applications with Tkinter
Input validation using try/except
Separating logic into reusable functions
Controlling widget states for a read-only output box

Tools Used

Python 3
Tkinter

How to Run
Download the file and run the following command in your terminal:
python MidtermProject1.py
